## Kotlin for JVM

#### Why Kotlin
- Pragmatic : 산업적인 용도의 언어 - 웹, 모바일, 데스크탑, 서버사이드 등 모두 가능하도록.
- Good Adaption in Android
  - boilerplate 를 많이 줄여주고
  - Java 와 호환 가능하다.
- C#, Java 와 유사함.

#### Introduction to the JVM
- 가장 흔한 구현은 open JDK
- an instance is what runs a java application
- applications compiled to bytecode
- JRE and JDK
